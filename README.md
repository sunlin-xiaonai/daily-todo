# Daily Todo

一个简单的「每日待办」单页应用。

## 在线预览

GitHub Pages（如果未开启，请在仓库 Settings → Pages 中启用）：

- https://sunlin-xiaonai.github.io/daily-todo/

## 本地打开

```bash
# 克隆仓库
git clone https://github.com/sunlin-xiaonai/daily-todo.git
cd daily-todo

# 用浏览器打开
open index.html  # macOS
# 或者手动在浏览器中打开 index.html
```

## 技术说明

- 纯 HTML + CSS + 原生 JavaScript
- 数据当前使用 `localStorage` 持久化在浏览器本地

更详细的设计与规范见：`SPEC.md`。
